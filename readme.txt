Protect Your Website WpAdmin. Modify page: Login, Logout, Forgot password, Registry, wp-admin.

== Description ==
If Your engine is Wordpress, You should use "Wp protect admin".

"Wp protect admin" helps customize URL admin panel and block default links.

<strong>IMPORTANT: Back up your ftp and database before beginning the activate plugin.</strong>. Sometimes the setting of other plug-ins can cause problems working our plugin.


<h4>Features</h4>
<ul>
<li>Define custom wp-login.php url(i.e <a href="http://your-domain.com/my-new-login)" rel="nofollow">http://your-domain.com/my-new-login)</a></li>
<li>Define custom wp-admin url(i.e <a href="http://your-domain.com/myadmin)" rel="nofollow">http://your-domain.com/myadmin)</a></li>
<li>Define custom forgot password url(i.e <a href="http://your-domain.com/my-new-admin)" rel="nofollow">http://your-domain.com/my-new-admin)</a></li>
<li>Define custom logout url(i.e <a href="http://your-domain.com/my-new-logout)" rel="nofollow">http://your-domain.com/my-new-logout)</a></li>
<li>Define custom register url(i.e <a href="http://your-domain.com/my-new-register)" rel="nofollow">http://your-domain.com/my-new-register)</a></li>
<li>Hide wp-login.php </li>
<li>Hide wp-admin</li>
</ul>



== Installation ==
<ol>
<li>Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your <code>wp-content/plugin</code> directory, or download from <a href="appsaur.co" title="Themes, plugins for Wordpress">appsaur.co</a>.</li>
<li>Activate the plugin through the WordPress admin panel.</li>
<li>Go to settings "Wp Protect Admin" and customize Your URL.</li>
<li>Done!</li>
</ol>



== Frequently Asked Questions ==
<ol>
<li> <strong>What i need do when I install plugin "Wp protect admin" ?</strong><br />

Please, create back up file and database, before change settings in "Wp protect admin". Sometimes the setting of other plug-ins can cause problems working our plugin.
</li>
<li> <strong>Nothing happen after change url wp-admin ?</strong><br />

Please, update the site permalink ("Settings" >> "Permalinks"), re-check and save.
</li>
<li> <strong>When i remove new admin url and save i see page 404. What can I do?</strong><br />

Please, click on admin page link dashboard. If still not work logout and login again.
</li>
<li> <strong>If i disable wp-login.php can I reset password?</strong><br />

If You hide wp-login.php, then You can't reset password.
</li>
</ol>
